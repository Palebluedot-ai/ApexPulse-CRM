## 4. 已完成代码

已扩展 confirm effects helper。

新增能力：

- 解析 `nextFollowupAt`
- 从 Review 自然字段构建新客户
- 从 Review 自然字段构建自动 followup 任务
- last contact 更新支持同步 `nextFollowupAt`

已扩展 Review confirm 后端流程。

确认记录时：

- 如果已有 partyId，则绑定已有客户
- 如果没有 partyId 但有客户名，则创建新客户
- confirmed event 会绑定最终客户
- 最终客户会刷新最近沟通状态
- 如果有 `nextAction`，会自动生成 followup 任务
- 如果有可解析的 `nextFollowupAt`，会写入客户和任务

已扩展 Review confirm API。

API 现在接收：

```text
naturalFields
```

已扩展 Review 前端提交。

点击“确认入库”时，前端会把自然字段传给后端。
