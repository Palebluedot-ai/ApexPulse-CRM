## 2. 已落地内容

新增 `src/lib/review-form.ts`。

它提供结构化字段解析能力：

- 空字符串解析成 `{}`
- 合法 JSON object 可以提交
- 非法 JSON 返回明确错误
- JSON array 会被拒绝

增强 `/review` 页面：

- 顶部显示待确认记录数量
- 每条记录显示“第 N 条”
- content type 显示为中文
- 每条记录显示来源 channel
- 每条记录显示附件数量
- 顶部增加客户列表入口
- 结构化字段下方增加 JSON object 提示
- edit 和 confirm 前先校验结构化字段
- 错误信息使用红色提示块
- 成功信息使用普通提示块
