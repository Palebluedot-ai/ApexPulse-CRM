## 4. 关键设计

`env:check` 会自动读取：

- `.env`
- `.env.local`
- 当前 shell 环境变量

优先级：

```text
shell 环境变量 > .env.local > .env
```

输出只显示 key 名，不显示任何 secret 值。

这样可以检查环境是否完整，同时避免把真实 key 打到终端输出或聊天记录里。
