## 4. 行动面板规则

行动面板只翻译已有跟进状态，不定义新分层。

当前映射：

- `overdue` -> 已经逾期
- `due_soon` -> 近期要跟进
- `unknown` -> 还没有跟进计划
- `up_to_date` -> 状态健康

当前不会自动改客户状态。

当前不会自动判定沉睡客户。

当前不会自动判定重点客户。
